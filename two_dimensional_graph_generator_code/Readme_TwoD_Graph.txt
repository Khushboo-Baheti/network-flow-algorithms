1. Copy program TwoD_Graph.java to eclipse.
2. Down load Jar files from the following link:
   https://drive.google.com/open?id=1UVsrqJmR9T6kqWDNOpJww3ztXyc01hPw
3. Add jar files to the project using the following steps:
        a) Right click on the eclipse project which you are working
        b) Go to properties--> Java Build Path --> Add External Jars
       c) Add all the 4 jar files 
4. Run the program TwoD_Graph.java. (change X and Y axis values as needed in main method in program
